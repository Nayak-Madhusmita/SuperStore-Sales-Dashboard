
<h1>🛒 Superstore Sales Analytics & Forecast Dashboard</h1>

<h2>📊 Project Overview</h2>
<p>
This project presents a <b>Sales Analytics and Forecasting Dashboard</b> built using <b>Power BI</b> based on the Superstore dataset.
The dashboard provides a comprehensive view of sales performance, profitability, customer segments, regional distribution, 
and future sales trends using time-series forecasting.
</p>

<p><b>Objective of the Project:</b></p>
<ul>
<li>Monitor sales performance across multiple dimensions</li>
<li>Identify high-performing regions and categories</li>
<li>Analyze profit trends and shipping patterns</li>
<li>Predict future sales using historical data</li>
</ul>

<hr>

<h2>🛠 Tools & Technologies Used</h2>
<ul>
<li><b>Power BI</b> – Data visualization and dashboard creation</li>
<li><b>Excel / CSV Dataset</b> – Data source</li>
<li><b>DAX</b> – Measures and calculated KPIs</li>
<li><b>Power Query</b> – Data cleaning and transformation</li>
<li><b>Power BI Forecasting Model</b> – 15-day sales prediction</li>
</ul>
